House Stock

House Stock es una aplicación para gestionar inventarios domésticos, permitiendo a los usuarios agregar, editar, eliminar y filtrar productos, así como escanear tickets para extraer datos mediante OCR.

Características

  • Gestión de Inventario: Agrega, edita y elimina productos en tu inventario.
  • Vista Detallada y Simple: Alterna entre vistas detalladas y simples del inventario.
  • Filtros y Ordenamiento: Filtra productos por nombre, tienda, cantidad, ubicación y fecha de compra. Ordena productos por nombre, precio, cantidad y fecha de compra.
  • Escaneo de Tickets: Escanea tickets para extraer automáticamente información del producto.
  • Autenticación: Registro, inicio de sesión y cierre de sesión de usuarios.

Instalación

  1. Clona el repositorio:
    git clone https://github.com/tu-usuario/house-stock.git
  2. Navega al directorio del proyecto:
    cd house-stock
  3. Instala las dependencias:
    npm install

Uso

  1. Inicia la aplicación:
    npm start
  2. Abre la aplicación en tu dispositivo o emulador usando Expo Go.

Scripts

  • npm start: Inicia la aplicación en modo desarrollo.
  • npm run build: Compila la aplicación para producción.
  • npm run lint: Ejecuta el linter para revisar el código.
  • npm run test: Ejecuta las pruebas.

Estructura del Proyecto

  • /src
    • /components
      • /nonReusable: Componentes específicos de la aplicación.
      • /reusable: Componentes reutilizables.
    • /context: Contextos para el manejo de estado global.
    • /hooks: Hooks personalizados.
    • /screens: Pantallas de la aplicación.
    • /types: Definiciones de tipos TypeScript.
    • /utils: Utilidades y funciones auxiliares.
